Config info

1) Change the App.config before deployment application
   debug=false to read log generated by MIM Hybrid Reporting. Set to true to parse the test PowerShell script
   logName="Identity Manager Request Log" don't change this if you want the program to read the event logs from MIM Hybrid Reporting
   logFile=[fully qualified filepath of where the text log is written. Leave as empty to disable file based logging
   workspaceId=[Log Analytics Workspace ID]
   workspaceKey=[Log Analytics security key]
   workspaceLogName="MIMHybrid" this can be any uniqie name

2) run the app in the administrator mode or in a credential that has permission to write to the Event Log.
